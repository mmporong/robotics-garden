Unity 게임 개발자에서 로보틱스 소프트웨어로 방향을 트는 과정의 학습 기록입니다. 부트캠프 커리큘럼(프로그래밍 기초 → ROS2 → 비전 → 딥러닝 → 모방학습 → 시뮬레이션 → 휴머노이드 캡스톤)을 따라가며, 배운 것을 그날 완결된 노트로 남깁니다.

## 이 가든을 읽는 법

- **노트는 서로 연결되어 있습니다.** 본문 링크와 오른쪽의 백링크·그래프로 앞뒤 맥락을 오갈 수 있습니다.
- **검색(Ctrl + K)** 으로 용어를 바로 찾을 수 있습니다.
- 모든 노트는 같은 원칙으로 씁니다 — 전문 용어를 본문으로 쓰고 바로 뒤에 풀이를 병기, 정의→원리→실전 증상 순서, "나중에 알아보기"로 미루지 않기.

## 무엇이 쌓이나

| 갈래 | 내용 |
|---|---|
| 학습 노트 | 수업·자습에서 배운 개념을 완결 서술로 정리 |
| 드릴 | 로봇 현업 시나리오(텔레메트리, 센서 버퍼, 검출 정렬)로 만든 연습 문제와 셀프체크 |
| 코드 리딩 | LeRobot 같은 오픈소스를 진도에 맞춰 대조하며 읽은 기록 |

노트는 원본 저장소에서 골라 발행하므로, 여기 있는 글은 모두 다시 읽을 것을 전제로 다듬어진 것들입니다.
